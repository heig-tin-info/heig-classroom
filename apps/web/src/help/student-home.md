# My assignments

## What this page shows

One card per classroom you belong to, with its assignments in a sortable table: deadline and live countdown, status, tests and grade. The search filters everything; the buttons switch between cards and a flat list.

## How to accept an assignment

Link your GitHub account first (Settings), then click **Accept**: your private repository is created and a GitHub invitation is sent — accept it to get access. Push your work there before the deadline.

## How grades work

While the assignment is open, every push runs the tests and shows an **indicative** grade. After the deadline your last commit is frozen and receives the full review; the final grade and a `GRADING.yml` with per-criterion points land in your repository.

## Teachers only

The graduation-cap button in the header switches back to the teacher view.
