# Branche à squasher

## Ce que ça choisit

La branche dont l'**arbre actuel** devient l'unique commit initial reçu par
les étudiants. Par défaut : la branche par défaut du dépôt ; le sélecteur
n'apparaît que si la source a plusieurs branches.

## Usage typique

Un dépôt canonique enseignant où `master` porte la solution qui fonctionne et
une branche `student` (ou similaire) porte le squelette distribué : choisissez
ici la branche du squelette.
