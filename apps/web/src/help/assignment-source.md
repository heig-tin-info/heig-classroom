# Source repository

## What it is

The repository holding the assignment material — statement, scaffold code,
tests, grading criteria. It must live in the classroom's organization.

## What happens to it

At creation the platform snapshots it into a `<slug>-squashed` repository
(see *Distributed source*); each student who accepts gets a private copy of
that snapshot. The source itself is never touched.

## Name tip

Picking a repository pre-fills the assignment name in humanized form
(`labo-02-quadratic` → “Labo 02 Quadratic”) — edit it freely; the slug of the
name becomes the prefix of every student repository.
