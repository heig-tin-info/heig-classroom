# Dépôt source

## Ce que c'est

Le dépôt qui contient le matériel du devoir — énoncé, code de départ, tests,
critères de correction. Il doit se trouver dans l'organisation de la classe.

## Ce qu'il devient

À la création, la plateforme le fige dans un dépôt `<slug>-squashed` (voir
*Distributed source*) ; chaque étudiant qui accepte reçoit une copie privée de
ce snapshot. Le dépôt source lui-même n'est jamais modifié.

## Astuce nom

Choisir un dépôt pré-remplit le nom du devoir en version lisible
(`labo-02-quadratic` → « Labo 02 Quadratic ») — modifiable librement ; le slug
du nom devient le préfixe de chaque dépôt étudiant.
