# Tâches planifiées

## Ce que c'est

La réconciliation d'arrière-plan avec GitHub. Les webhooks traitent tout en temps réel ; ces tâches sont le **filet de sécurité** qui rattrape les livraisons perdues (pushes ou runs manqués).

## Comment les piloter

Les intervalles sont éditables, une tâche peut être mise en pause, et **Run now** en déclenche une immédiatement. Un échec est retenté à la passe suivante — rien n'est perdu.
