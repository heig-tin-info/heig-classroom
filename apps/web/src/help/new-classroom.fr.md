# Nouvelle classe

## Ce qu'il faut

Une **organisation GitHub** pour héberger les dépôts étudiants. Une classe = une organisation + une liste d'étudiants.

## Comment la créer

Choisissez une organisation où la GitHub App est déjà installée, ou saisissez un autre nom d'organisation. Elle doit exister sur GitHub ; vous pourrez installer l'App depuis la page de la classe ensuite — les devoirs en ont besoin.
