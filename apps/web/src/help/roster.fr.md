# Roster

## Ce qu'est cette liste

La liste de classe. Chaque ligne est un siège : **pending** tant que l'étudiant ne s'est pas connecté avec l'e-mail correspondant, **claimed** une fois rattaché à son compte. Un badge *staff* marque un siège enseignant (voir plus bas).

## Comment marche le rattachement

Les étudiants rattachent leur siège automatiquement à la première connexion — pas de code d'invitation. En cas de conflit (même étudiant, deux sièges), la ligne est signalée pour que vous tranchiez.

## Gérer les lignes

Les icônes à droite éditent une ligne, révoquent un rattachement ou retirent l'étudiant. L'enveloppe ouvre votre client mail. Changer un e-mail révoque le rattachement de la ligne.

## Rejoindre votre propre classe

**Me joindre comme étudiant** vous donne un siège *staff*, exclu de l'effectif : combiné au bouton chapeau de diplômé dans l'en-tête, vous parcourez tout le flux étudiant (accepter, pousser, notes) sans second compte.
