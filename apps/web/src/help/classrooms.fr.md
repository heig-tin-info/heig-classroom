# Mes classes

## Qu'est-ce qu'une classe

Une classe lie une **organisation GitHub** à une **liste d'étudiants**. Les devoirs créés dedans sont distribués sous forme de dépôts privés dans cette organisation, un par étudiant.

## Changer de vue

Les boutons à droite basculent entre **cartes**, **liste triable** et **frise chronologique**. Le champ de recherche filtre à la frappe (nom et organisation, correspondance floue).

## Ouvrir les choses

- Cliquez une carte pour ouvrir la classe.
- Cliquez le nom de l'organisation pour l'ouvrir sur GitHub.
- Survolez les badges étudiants pour voir qui est inscrit et qui a déjà rattaché son compte.

## Voir les classes archivées

Cliquez le bouton **boîte d'archive** à côté des boutons de vue. Les classes archivées apparaissent en cartes atténuées avec leur date d'archivage ; **Restaurer** en ramène une parmi les classes actives. Archiver ne supprime rien — données et dépôts GitHub sont conservés.

## Tester l'expérience étudiante

Ouvrez une classe et utilisez **Me joindre comme étudiant** sur la carte Roster : vous prenez un siège *staff* (exclu de l'effectif). Le bouton **chapeau de diplômé** dans l'en-tête bascule alors vers la vraie vue étudiant — acceptez le devoir, poussez, suivez votre note.
