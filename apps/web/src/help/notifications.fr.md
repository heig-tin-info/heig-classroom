# Notifications

## Toasts dans l'application

Des toasts temps réel apparaissent en bas à gauche quand il se passe quelque chose dans vos classes : un étudiant rejoint, accepte un devoir, pousse, une note est capturée. Activez chaque type ici — ce réglage est stocké **dans ce navigateur**.

## Notifications par e-mail

La carte suivante configure les e-mails (nouveau devoir, rappel d'échéance, note finale, échecs de provisioning…). Ces préférences sont stockées **sur votre compte** et vous suivent d'un appareil à l'autre ; chaque message contient aussi un lien de désinscription en un clic pour son type.
